# EonSwap Documentation

EonSwap is a non-custodial platform for token swaps and cross-chain transfers.

Use this documentation to understand the product, reduce mistakes, and trade with confidence.

## What You Can Do

- Swap tokens with aggregated routing
- Bridge assets across supported networks
- Track transaction progress and outcomes
- Check system status during high traffic
