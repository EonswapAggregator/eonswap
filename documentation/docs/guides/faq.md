# FAQ

## Is EonSwap custodial?

No. EonSwap is non-custodial.

## Can I use EonSwap for both swap and bridge?

Yes. EonSwap supports both flows in one interface.

## Why is there a delay in bridge completion?

Bridge finalization depends on source/destination chain conditions and provider processing.

## Is every transaction guaranteed to succeed?

No. Success depends on liquidity, network state, gas, and transaction conditions.

## Where can I check my transaction progress?

Use Activity and Status pages.
