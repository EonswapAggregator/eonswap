# Troubleshooting

## Quote not loading

- Check wallet network.
- Retry with a different amount.
- Wait and retry during heavy market activity.

## Bridge seems slow

- Cross-chain transfers can take longer during congestion.
- Keep transaction hash and monitor status updates.

## Transaction failed

- Check available gas token balance.
- Re-check amount and token pair.
- Retry after network conditions improve.

## Status shows degraded

- Degraded status may be temporary.
- Retry later if provider traffic is high.
